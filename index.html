ALLE(le)S

By Kenza Samlali

About

Alle(le)s is an interactive web app
that allows you to experience the basic
building blocks of life: nucleotides.
Computers communicate in 1's and 0's.
The code of living things is A's, C's, G's and T's
A huge database called NCBI contains the genetic information
of millions of living organisms, written in this code
Use this WebApp to search it,
and to draw out a string of nucleotides
to experience the length of these sequences

Instructions:


Search the nucleotide database for an animal, tissue, cell line, gene...
Example: mus musculus
Drag to draw DNA
Drag Two fingers to make RNA
